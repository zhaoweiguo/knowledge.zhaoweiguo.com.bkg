实打实的 Go 项目，但是 Github 却统计为 JavaScript 怎么办::

    原理:
    Github 采用 Linguist 来自动识别你的代码，然后判断属于哪一种语言。

    解决方法:
    1. 在项目中增加更多的你所想要归属的语言的代码（综合性项目，最好是采用这种方式）。
    2. 在项目中添加 .gitattributes:
      *.js linguist-language=go
      *.java linguist-language=go
