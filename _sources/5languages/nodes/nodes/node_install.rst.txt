node简介
#######################

* 官网: http://nodejs.org


node升级::

    //node有一个模块叫 n ，是专门用来管理node.js的版本的。
    npm install -g n
    n stable    // 升级node.js到最新稳定版
    or
    n v0.10.26  // 升级到指定版本
    

* 启动与依赖::

    npm install nodemon -g   // 使用命令nodemon app.js, 保存时自动加载
    npm install -d    // 使用package.json文件
